open controller

  • 공통 Open Controller은 MVC View Resolver 를 통해 view 페이지를 열게 된다.

  • 형식 : [폴더명]/[파일명](확장자제외) 혹은 [폴더명]/[폴더명]/[파일명]
    Thymeleaf view 파일 기본 폴더인 templates 하위를 기준 경로 (확장자는 application.properties에 선언되어 있습니다)
  • 예) templates/test/test_list.html 파일을 여는 경우
  • URL : /open/test/test_list
  • /open/test/test_list -> SangsCommonOpenController class -> test/test_list.html
