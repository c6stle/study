excel down controller

  • 공통 Exceldown Controller은 Service Class 처리 후 Workbook 타입으로 반환받은 엑셀문서를 다운받도록 합니다.

    형식 : /exceldown/[서비스ID]/[메서드ID]

    서비스ID : Service Class 의 ID 값 -> 서비스클래스의 첫번째 글자를 소문자로 하고 클래스명의 Service부분을 제거한 ID 값 예) SampleService -> sample, UserMngrService -> userMngr
    메서드ID : 메서드명
  • 예) /view/sample/testList
  • URL : /exceldown/sample/testExcelDown
  • /exceldown/sample/testList -> SangsCommonExcelDownController -> SampleService.class의 매서드 testExcelDown 호출 -> SangsCommonExcelDownController 에서 byte stream 으로 다운로드 실행
